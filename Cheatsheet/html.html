Stucture HTML

Add title in the tap bar

  • Always start the html code with <!DOCTYPE html > .
  • Then add the start tag <head>and add under it a new start tag <title> where we can add a title to the tap bar in your browser
  • Then add the end tag </title>
  • In the next line, close the head

Add the body

Definition: The body will be the content of the document

  • Open the tag <body>
  • In tne next line you can add a heading tag <h1> . Definition: HTML defines six levels of headings tag. The heading elements are H1, H2, H3, H4, H5, and H6 with H1 being the highest or most important level and H6 the least tag.

Add paragraph

We can add paragraphs by openning the tag <p> .
